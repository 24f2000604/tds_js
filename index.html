Risk Assessment Model (Python)

Historic Value at Risk (VaR) calculation.

var_calculator.py

# Author: 24f2000604@ds.study.iitm.ac.in
import numpy as np
import pandas as pd

def calculate_var(returns, confidence_level=0.95):
    """Calculates Historic Value at Risk (VaR)"""
    sorted_returns = np.sort(returns)
    var_index = int(len(sorted_returns) * (1 - confidence_level))
    return sorted_returns[var_index]

# Usage
portfolio_returns = np.random.normal(0.001, 0.02, 252)
var_95 = calculate_var(portfolio_returns, 0.95)
print(f"95% VaR: {var_95:.4f}")

// Real-time market data streaming
const WebSocket = require('ws');

class MarketDataStream {
  constructor(url) {
    this.ws = new WebSocket(url);
    this.subscribers = [];
  }

  subscribe(callback) {
    this.subscribers.push(callback);
    this.ws.on('message', (data) => {
      callback(JSON.parse(data));
    });
  }
}

const stream = new MarketDataStream('wss://market.example.com');
stream.subscribe(data => console.log('Price:', data.price));

Data Analytics Query (SQL)

Quarterly revenue aggregation.

quarterly_report.sql

-- Author: 24f2000604@ds.study.iitm.ac.in
-- Quarterly Revenue Analysis
-- Financial Institution Data Warehouse

SELECT 
    segment_name,
    SUM(revenue) AS total_revenue,
    AVG(margin) AS avg_margin,
    COUNT(DISTINCT customer_id) AS unique_customers,
    ROUND(SUM(revenue) / LAG(SUM(revenue)) 
        OVER (ORDER BY quarter) - 1, 4) AS growth_rate
FROM financial_transactions
WHERE quarter = 'Q3-2025'
GROUP BY segment_name
ORDER BY total_revenue DESC;

Capital Asset Pricing Model (CAPM)

Expected return considering systematic risk (Œ≤).

$$E(R_i) = R_f + \beta_i \cdot (E(R_m) - R_f)$$

Where: $R_f$ = risk-free rate, $\beta_i$ = asset beta, $E(R_m) - R_f$ = market risk premium

Example: $R_f = 3\%$, $\beta = 1.2$, $E(R_m) = 10\%$ ‚Üí $E(R_i) = 11.4\%$

Sharpe Ratio

Risk-adjusted return measurement.

$$S = \frac{R_p - R_f}{\sigma_p}$$
  • $R_p$ = Portfolio return
  • $R_f$ = Risk-free rate
  • $\sigma_p$ = Portfolio standard deviation

Our Q3 Sharpe: $S = 1.85$

Black-Scholes Option Pricing

European option valuation model.

$$C = S_0 N(d_1) - K e^{-rT} N(d_2)$$
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$ $$d_2 = d_1 - \sigma\sqrt{T}$$

$C$ = Call price, $S_0$ = Stock price, $K$ = Strike, $r$ = Rate, $T$ = Time, $\sigma$ = Volatility

Compound Interest Formula

Investment growth projections.

$$A = P \left(1 + \frac{r}{n}\right)^{nt}$$

Continuous compounding: $A = Pe^{rt}$

Example: $P = \$100{,}000$, $r = 8\%$, $n = 12$, $t = 5$ years ‚Üí $A = \$148{,}985$

Financial Mathematics Summary

Present Value: $PV = \frac{FV}{(1+r)^n}$

Net Present Value: $NPV = \sum_{t=0}^{n} \frac{C_t}{(1+r)^t}$

IRR: $0 = \sum_{t=0}^{n} \frac{C_t}{(1+IRR)^t}$

Beta: $\beta = \frac{Cov(R_i, R_m)}{Var(R_m)}$

Standard Deviation: $\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \mu)^2}$
