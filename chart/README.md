## Business Context

Customer journey mapping and conversion funnel analysis for executive presentation.

This Alluvial Diagram visualizes how customers flow through different stages of the sales funnel across marketing channels.

## Files in This Repository

- `README.md` - This file (contains email: 24f2000604@ds.study.iitm.ac.in)
- `chart.png` - RAWGraphs generated Alluvial Diagram (512x512 pixels)
- `data.csv` - Source data for the visualization

## Data Structure

| Field | Description |
|-------|-------------|
| Channel | Marketing channel (Social, Email, Organic, Paid) |
| Stage | Funnel stage (Awareness, Interest, Decision, Purchase) |
| Outcome | Final result (Converted, Abandoned, Pending) |
| Count | Number of customers |

## How to Recreate in RAWGraphs

1. Visit [RAWGraphs](https://rawgraphs.io/)
2. Copy the data from `data.csv` below
3. Paste into RAWGraphs data input
4. Select **Alluvial Diagram** chart type
5. Map dimensions:
   - **Steps:** Channel → Stage → Outcome
   - **Size:** Count
6. Customize colors and labels
7. Export as PNG (512x512)
